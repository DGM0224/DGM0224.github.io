팀 성과

OneStack은 팀원들과의 협업으로 개발한 첫 서비스입니다.
실무에서 사용되는 Spring Boot 기반의 API 설계 및 DB 연동을 통해 백엔드 역량을 강화하였으며,
Jira, Confluence 기반 협업으로 기획 → 개발 → 배포까지 전체 사이클을 경험했습니다.

개인 기여

본 프로젝트에서 백엔드 개발자로서 주요 기능 개발과 인프라 구축에 깊이 관여하였습니다.

MySQL을 기반으로 초기 데이터베이스 스키마 설계 및 관리를 주도하였으며,
정규화를 통해 데이터 중복을 최소화하고, ERD 작성 및 인덱스 최적화로 성능 개선

Spring Boot와 MyBatis 기반으로 전문가 심사 체크리스트 시스템을 설계

Ajax 기반 실시간 데이터 조회 기능과 IntersectionObserver API를 활용한 무한 스크롤 구현으로
사용자의 검색 조건에 따른 결과를 빠르고 효율적으로 제공

전문가 검색 시 필터링 조건에 따른 평균 외주 가격을 실시간 산출하는 기능을 개발해
사용자가 자신의 조건에 맞는 합리적인 외주 비용을 빠르게 예측 가능

전문가의 프로필 상세 페이지 및 견적 요청 기능을 구현하여 REST API 기반으로 클라이언트-서버 간 데이터를 안정적으로 처리

실시간 채팅 기능 내에서 포트원 API를 연동하여 결제 기능을 개발하고, 결제 내역 상태를 직관적으로 확인할 수 있는 사용자 페이지 제공

프로젝트 완료 시점에 전문가 정보 자동 업데이트 기능을 개발해 고용횟수, 별점 등 수동 갱신 없이 자동반영해 운영 효율성 증가

전문가 프로필에 신고 기능을 구현하여, JavaScript를 통한 서버 연동으로 부적절 사용자에 대한 모니터링 기능 제공

AWS EC2를 이용한 서비스 배포 및 Nginx 기반 정적 파일 CDN 구성
또한 Docker + Jenkins로 CI/CD 파이프라인을 구축하여 배포 자동화와 개발 효율성 향상

이 프로젝트를 통해 백엔드 실무 능력을 체득하였고,
클라우드 기반 배포, DevOps 환경 구성, 데이터 흐름 최적화 등 다양한 기술을 경험하며
향후 DevOps 및 데이터 엔지니어링 분야로의 성장을 도모하고 있습니다.

트러블슈팅 & 해결

트러블 슈팅 및 해결 방안

비동기 무한스크롤 로딩 중 중복 데이터 표시

스크롤을 빠르게 내리면 같은 데이터가 두 번씩 출력되는 문제 발생
이를 해결하기 위해, 이미 조회된 데이터를 제외하고 조회하는 쿼리로 수정 및 API 요청 중엔 추가 요청을 막는 로딩 스피너 사용

결제 금액 조작 방지 미비

프론트에서 가격을 변경해 결제 요청을 보내는 악용 시도 발견
이를 해결하기 위해, 백엔드에서 서버 기준으로 금액 검증하는 로직을 추가

결제 시스템 오류

결제 시스템에서 발생한 결제 실패 문제는 외부 API와의 통신 오류로 인해 발생
이를 해결하기 위해, 결제 처리 과정에서 예외 처리를 강화하고, 결제 실패 시 모든 과정을 취소하는 트랜잭션을 추가하여 안정성을 확보

정규화 부족으로 인한 데이터 중복

전문가 정보가 여러 테이블에 중복 저장되어 관리 어려움 발생
이를 해결하기 위해, 전문가 프로필 정보를 하나의 테이블로 분리하고, 다른 테이블은 외래키 참조로 연결해 3정규형 기반으로 재설계

Nginx에서 정적 파일 미출력 (CDN 서버 문제)

이미지가 로컬에선 잘 나오는데, 배포 서버에선 404 오류 발생
Nginx에서 root 경로 설정이 잘못되어 있던 문제 확인
이를 해결하기 위해, location 블록 수정 및 권한 문제 해결로 정상 동작

Jenkins 빌드 실패

GitHub에서 push하면 Jenkins 빌드가 자동 실행되지만, 종종 실패 발생
빌드 시점에 사용 중인 포트가 열려 있어 컨테이너 충돌 발생 문제 확인
이를 해결하기 위해, 기존 컨테이너를 중지하고 새로 빌드하도록 파이프라인 수정
